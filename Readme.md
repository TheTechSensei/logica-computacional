## Get started 🚙

### Prerequisites
- [GHCi](https://downloads.haskell.org/ghc/latest/docs/users_guide/ghci.html)


### Installation

  ```sh
    sudo apt install ghc
    git clone git@github.com:Yony6041/logica-computacional.git
    cd logica-computacional
  ```

### Run the program 

  ```sh
      ghci practica2.hs
  ```
